Unlike traditional risk models that treat failure as a homogeneous event, StARS decomposes systemic misalignment into two chemically distinct vectors:
1.  **Agent Stress Load (ASL):** The entropic load carried by the intelligent agent (burnout, moral injury, computational fatigue).
2.  **Code Vulnerability (CV):** The structural fragility of the governing system (protocol complexity, conflicting KPIs, policy gaps).

This repository implements the **Mena Dominance Law**, a fundamental conservation law for organizational stability:

> **$\Delta = ASL - CV$**

The system calculates a **Dominance Differential ($\Delta$)** to enforce a **Dual-Path Corrective Mandate**:
* **Structure-Dominant ($CV > ASL$):** Deviation is driven by architecture. Corrective action MUST target the structure (e.g., protocol simplification). Targeting the agent increases entropy.
* **Agent-Dominant ($ASL > CV$):** Deviation is driven by the agent. Corrective action MUST stabilize the agent (e.g., workload reduction).
* **Mixed Dominance ($ASL \approx CV$):** Requires synchronized dual-path intervention.
